# How do I sign up to Sherpa as an admin user?

Please sign up to Sherpa in the normal way via the link <https://sherpa.app.escape-technology.com/signup>

You will then need to contact the sales team via the **Contact us** link at <https://sherpa.escape-technology.com/>.

You will be sent the Sherpa Terms and Conditions. When we have received your completed documents we will create a Sherpa account and admin login for you to enjoy.
